# Use
This app was built to navigate the World Heritage Sites database.

# Setup
1. Clone the repository to local machine
2. Open the project
3. `pip install venv`
4. `virtualenv venv`
5. `source venv/bin/activate` (Linux/Mac) or `venv\Scripts\activate` (Windows)
6. `pip install -r requirements.txt`
7. Run the app using `python server.py`
